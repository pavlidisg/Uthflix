To run this project use xampp (local server). Open sql admin (phpMyAdmin) and create a databse named db2. Import the sql files there. After user accounts are created, manually set privileges for the 4 user accounts needed for this poject to db2.

This project is focusing on database rather than UI. 
features: 
1. login / register
2. shows movies and series info for users and allowing them to comment and rate them , storing data in database.
3. producers can add movies/series or edit existing movie/series info.
